# 🧬 Envision 2026 — MD Simulator Tasks

Welcome to the **Envision 2026 Molecular Dynamics Simulator Tasks** repository!  
This is your hands-on space to learn GitHub workflows while working with simulation-related files — including future LAMMPS projects.

---

## 📋 Task List

### ✅ Task 1 — Fork & Clone the Repo
> Learn how to copy a repo to your account and get it on your machine.

1. Click **Fork** (top-right of this page)
2. Clone your fork:
```bash
   git clone https://github.com/YOUR_USERNAME/Envision_2026_MD_Simulator_Tasks.git
   cd Envision_2026_MD_Simulator_Tasks
```

---

### ✅ Task 2 — Create Your Branch
> Branches let you work without affecting the main project.
```bash
git checkout -b add-YOUR_NAME
```
Example: `git checkout -b add-Skanda`

---

### ✅ Task 3 — Add Your Entry
> Make your first real contribution!

1. Open the file `participants.md`
2. Add a new line in this format:
```
   | Your Name | Your Institute | Favourite Physics Topic |
```
3. Save the file

---

### ✅ Task 4 — Commit & Push
> Save your changes and upload them to GitHub.
```bash
git add participants.md
git commit -m "Add YOUR_NAME to participants list"
git push origin add-YOUR_NAME
```

---

### ✅ Task 5 — Open a Pull Request
> This is how real-world collaboration works!

1. Go to your fork on GitHub
2. Click **"Compare & pull request"**
3. Add a short description of what you did
4. Click **"Create pull request"** 🎉

---

### ✅ Task 6 — Review a Peer's PR
> Leave a comment or suggestion on someone else's pull request.

- Go to the **Pull Requests** tab
- Open any open PR
- Leave a constructive comment or approve it

---

### ✅ Task 7 — Open an Issue
> Issues are how teams track bugs and ideas.

- Go to the **Issues** tab → **New Issue**
- Suggest a new task, report something unclear, or propose a LAMMPS file structure
